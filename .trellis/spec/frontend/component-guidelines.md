# Component Guidelines

> Not applicable — this is a Rust project.

---

## Overview

This project is a Rust workspace (`quanergy_client_rs`) with a core SDK library
and CLI binary apps. There is no web application frontend, no React/Vue/Svelte
components, and no component framework.

The only "frontend" artifact is the VitePress documentation site under `docs/`,
which is covered by [Directory Structure](./directory-structure.md).

Do not attempt to match React/Vue component patterns in this project — they
do not apply.
